HTML 行内标签

a标签: a 标签定义超链接，常用于网页之间的跳转、设置文档书签，以及链接电话、邮箱等。

语法示例: <a href="链接地址"></a>


span标签: span标签用于组合文档中的行内元素，它就像一个透明的盒子，可以把多个盒子放在架子的同一行。

语法示例: <span></span>


strong标签: strong标签用于给文字加粗

语法示例: <strong></strong>


img标签: img标签用于展示图片

语法示例: <img src="图像地址" alt="图像描述" />

HTML 块级标签

div标签: div标签是最重要的一个块级元素，它可以是组合其他 HTML 元素的容器，相当于是个透明的盒子，可以把其他标签放在里面。

语法示例: <div></div>


p标签: p标签是用来划分 HTML 文档段落的。

语法示例: <p></p>


标题标签: 在 HTML 中一共有六级标题，用 <h1> ~ <h6> 来表示，其中 <h1> 的标题字号最大，依次递减。

语法示例: <h1></h1>
